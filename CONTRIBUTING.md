# Contributing to Port Docs

Thank you for your interest in contributing to Port's documentation!

We value your contributions and want to make the process as easy as possible.

## Getting Started

1.  **Installation**: Follow the [Installation instructions](./README.md#installation) in the README to set up the project locally.
2.  **Style Guide**: Please read our [Documentation Style Guide](./best_practices.md) before writing any content. It covers:
    *   Tone and Style
    *   Formatting (Tabs, Links, Lists)
    *   Images & GIFs
    *   Code examples

## How to Contribute

1.  **Fork** the repository.
2.  Create a new **branch** for your feature or fix.
3.  Make your changes.
4.  Run `npm start` to preview your changes locally.
5.  Submit a **Pull Request**.

## Questions?

If you have any questions, feel free to open an issue or reach out to us.
