:::tip Available GitLab resources

The following resources can be used to map data from GitLab, it is possible to reference any field that appears in the API responses linked below for the GitLab integration mapping configuration.

- [`group`](https://docs.gitlab.com/ee/api/groups.html#list-groups)
- [`project`](https://docs.gitlab.com/ee/api/groups.html#list-a-groups-projects)
- [`merge-request`](https://docs.gitlab.com/ee/api/merge_requests.html#list-project-merge-requests)
- [`pipeline`](https://docs.gitlab.com/ee/api/pipelines.html#list-project-pipelines)
- [`job`](https://docs.gitlab.com/ee/api/jobs.html#list-project-jobs)
- [`issue`](https://docs.gitlab.com/ee/api/issues.html#list-project-issues)

:::
