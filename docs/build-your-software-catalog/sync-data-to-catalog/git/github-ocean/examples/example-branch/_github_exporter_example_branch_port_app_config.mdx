Use `branchNames` to ingest only specific branches. When you set it, the integration fetches those branches explicitly instead of listing all branches using pagination.

```yaml showLineNumbers

  selector:
    query: "true"
    # highlight-next-line
    branchNames: ["main", "develop"]
```
