<h2>Installation</h2>

To install the integration, follow theses steps:

1. Go to the <a href={`https://app.getport.io/settings/data-sources?section=EXPORTERS&provider=${props.integration}`}>{props.integration} data source page</a> in your portal.

2. Select the `Real-time and always on` method:
    <img src="/img/sync-data-to-catalog/selfHostedMethod.png" width="40%" border='1px' />

3. A `helm` command will be displayed, with default values already filled out (e.g. your Port cliend ID, client secret, etc).  
Copy the command, replace the placeholders with your values, then run it in your terminal to install the integration.
