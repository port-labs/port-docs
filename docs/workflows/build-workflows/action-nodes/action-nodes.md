# Action nodes

<ClosedBetaFeatureNotice id="workflows" />

Action nodes perform operations in your workflow. They execute after trigger nodes and can be chained together to create complex automation flows.

## Available action types

| Action type | Description |
| ----------- | ----------- |
| [Webhook](/workflows/build-workflows/action-nodes/webhook) | Send HTTP requests to external endpoints |
| [Upsert entity](/workflows/build-workflows/action-nodes/upsert-entity) | Create or update entities in your software catalog |
| [Kafka](/workflows/build-workflows/action-nodes/kafka) | Publish messages to Port's Kafka topic |
| [Integration actions](/workflows/build-workflows/action-nodes/integration-actions/) | Trigger actions in integrated platforms like GitHub |
