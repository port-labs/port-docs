:::tip blueprint dependencies
This blueprint also has an aggregation property configured for the `statuspageComponent` blueprint.
Since the `statuspageComponent` blueprint depends on the `statuspage` blueprint, 
you will need to initially create the `statuspage` blueprint without the aggregation property, 
and then add it once the `statuspageComponent` blueprint exists.
:::
